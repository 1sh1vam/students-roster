## Getting Started

### Prerequisites

- Node.js
- NPM

### Installation and setup

1. Clone the repository:

   ```bash
   git clone https://github.com/1sh1vam/students-roster.git
2. Setup the .env in /server folder with DB_URI
3. Run `cd server && npm start`
4. Run `cd client && npm run dev`
